Week 1

College: webtechnologie

Hoe werkt het internet? Je apparaat stuurt een HTTP Request naar de cloud, deze kijkt vervolgens in de server. De server kijkt met het gekregen adres naar de bijbehorende bestanden. De bestanden worden vervolgens in een HTTP Response terug gestuurd naar je apparaat.

HTML

HTML is het skelet van de pagina, hier komt alle informatie en content in. Het werkt op basis van de DOM (Document Object Model). Het is bij HTML van belang dat je de elementen nest op de correcte manier. Een <h1> kan bijvoorbeeld goed in een <header>, maar een <main> kan niet in een <p>.

CSS

CSS is de huid van de website. Het bepaald hoe de website er uit komt te zien. In CSS (Cascading Style Sheet) werk je met verschillende selectoren om het juiste element te wijzigen.

  • *
  • .class
  • #id
  • p
  • p, h1
  • p h1
  • p > h1
  • p + h1
  • p ~ h1
  • a:link
  • :[target]
  • a[href:]
  • p::after
  • p::before
  • p:nth-of-type

JS

Javascript is voor het gedrag van de website.

Wat is het verschil tussen Flexbox Layout en Grid Layout? Wanneer zou je welke gebruiken?

Flex box is op 1 as, dus x of y en daar mee kan je flexibel elementen plaatsen. Grid is voor op 2 assen dus voor zowel de x als y-as.

Wat is de default Layout Mode van elk element?

Flow Layout

Schrijf op welke Layout Modes je per onderdeel nodig hebt voor het ontwerp van de opdrachtgever.

Voor de navigatie bar is Flex layout het beste, voor de online catalogus is grid het beste.

Voor wie is toegankelijkheid belangrijk?

Toegankelijkheid is eigenlijk voor iedereen belangrijk. Of je nou blind bent, doof bent, geen armen hebt, of een beperking voor korte termijn. Het is belangrijk dat elke gebruiker je website goed kan bedienen.

Wat moet ik voor woensdag nog doen voor de audit?

Ik moet deze stap nog doen: Test de interactive elements en headings and landmarks van de website, zoals beschreven in het artikel “How To Do an Accessibility Review”. Ook moet ik nog de ReadMe afmaken waar ik samenvattend bescrhijf wat er veranderd zou moeten worden.
